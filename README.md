## 🔧 Customization Guide

### Changing Colors
```css
/* Update the main red color */
:root {
  --netflix-red: rgb(229, 9, 20);
  --netflix-black: rgb(0, 0, 0);
  --netflix-gray: rgb(46, 44, 44);
}
```

### Adding New Sections
```html
<!-- Follow this structure for new sections -->
<section class="first">
    <div class="sechead">
        <h1>Your Title</h1>
        Your description text here.
    </div>
    <div class="secimg">
        <img src="your-image.jpg" alt="Description">
    </div>
</section>
```
